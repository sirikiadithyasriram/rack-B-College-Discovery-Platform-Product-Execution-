# College Platform Frontend

A clean, minimal Next.js 14 + Tailwind frontend for the college platform.

## Setup

1. Install dependencies:

```bash
cd "c:\New folder\college-platform-frontend"
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open the app at `http://localhost:3000`.

## Environment

Create `.env.local` with:

```env
NEXT_PUBLIC_API_URL=http://localhost:5000
```

## Features

- College listing with search and filters
- College detail page
- Compare page
- Login / registration pages
- Saved colleges page
- Clean responsive UI inspired by minimal education apps
